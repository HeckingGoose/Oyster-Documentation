# For Game Developers

All pages in this section are targetted towards developers who are looking to use Oyster within the games they make.

It will cover:
- Important things to know when adding Oyster to a game,
- How to write a script,
- Commands that are built-in to Oyster,
- Commands that are supported by specific games that use Oyster.
